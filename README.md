# customer-churn-analsis-SQL-EXCEL
Customer Churn Analysis using SQL and Excel with dashboard and business insights

## Objective
To analyze customer churn behavior and identify key factors contributing to customer attrition.

## Tools Used
- SQL
- Microsoft Excel

## Key Analysis
- Overall churn rate calculation
- Contract type vs churn analysis
- Tenure impact on churn
- Monthly charges vs churn
- Revenue at risk estimation

## Key Insights
- Month-to-month contract customers have the highest churn
- Customers with shorter tenure are more likely to churn
- Higher monthly charges are associated with increased churn

## Outcome
Created an Excel dashboard to visualize churn KPIs and support business decision-making.
